# Фотографии «До/После»

Разместите в этой папке реальные фотографии пациентов в формате .webp (рекомендуется).

## Необходимые файлы:

- botox-before.webp
- botox-after.webp
- lips-before.webp
- lips-after.webp
- smas-before.webp
- smas-after.webp
- ipl-before.webp
- ipl-after.webp
- prx-before.webp
- prx-after.webp
- aptos-before.webp
- aptos-after.webp
- cleaning-before.webp
- cleaning-after.webp
- biorev-before.webp
- biorev-after.webp

## Рекомендации:
- Соотношение сторон: 4:5 (портретная ориентация)
- Разрешение: 800×1000px (достаточно для веба)
- Формат: WebP (сжатие 80-85%)
